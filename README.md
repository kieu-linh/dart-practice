## Installation
1. Clone the repository:

    ```
    git@github.com:kieu-linh/dart-practice.git
    ```
2. Checkout branch:

    ```
    git checkout <feature/practice-dart>  
    ```
3. Pull origin branch:

    ```
    git pull origin <feature/practice-dart> 
    ```
4. Run project
   ```
   dart run
   ```
